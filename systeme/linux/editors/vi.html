Basic vi Commands

Démarrer et quiiter vi

* vi filename edit filename starting at line 1
  vi -r filename recover filename that was being edited when system crashed

Quitter vi

* :x<Return> commande équivalente à ":wq"
  :wq<Return> sauvegarde les modifications et quitte le fichier
  :q<Return> quitter le fichier
* :q!<Return> quitter le fichier sans sauvegarder

Configuration VI

* :set nu affiche le numéro des lignes
  :set nonu désactive l'affichage du numéro des lignes

Navigation dans VI

* j or <Return>
  [or down-arrow] 		se déplacer vers le bas
* k [or up-arrow] se déplacer vers le haut
* h or <Backspace>
  [or left-arrow] 		se déplacer vers la gauche
* l or <Space>
  [or right-arrow] 		se déplacer vers la droite
* 0 (zero) déplacer le curseur au début de la ligne courante
* $ déplacer le curseur à la fin de la ligne courante
  w déplacer le curseur au début du mot suivant
  b déplacer le curseur au début du mot précédent
  :0<Return> or 1G déplacer le curseur au début de la ligne
  :n<Return> or nG déplacer le curseur à la ligne n
  :$<Return> or G déplacer le curseur à la dernière ligne du fichier

Screen Manipulation

The following commands allow the vi editor screen (or window) to move up or down several lines and to be refreshed.

  ^f move forward one screen
  ^b move backward one screen
  ^d move down (forward) one half screen
  ^u move up (back) one half screen
  ^l redraws the screen
  ^r redraws the screen, removing deleted lines

Ajouter, Changer, et Supprimer du texte

* u annule la dernière action effectuée

The main purpose of an editor is to create, add, or modify text for a file.

Insérer ou ajouter du texte

* i insérer du texte à l'endroit où se situe le curseur jusqu'à l'action <Esc>
  I insérer du texte au début de la ligne jusqu'à l'action <Esc>
* a insérer (append) du texte après le curseur, jusqu'à l'action <Esc>t
  A insérer du texte à la fin de la ligne, jusqu'à l'action <Esc>
* o créer (open) et met le texte en dessous de la ligne courante, jusqu'à l'action <Esc>
* O créer (open) au dessus de la ligne courante, jusqu'à l'action <Esc>

Changing Text

The following commands allow you to modify text.

* r replace single character under cursor (no <Esc> needed)
  R replace characters, starting with current cursor position, until <Esc> hit
  cw change the current word with new text,
starting with the character under cursor, until <Esc> hit
  cNw change N words beginning with character under cursor, until <Esc> hit;
  e.g., c5w changes 5 words
  C change (replace) the characters in the current line, until <Esc> hit
  cc change (replace) the entire current line, stopping when <Esc> is hit
  Ncc or cNc change (replace) the next N lines, starting with the current line,
stopping when <Esc> is hit

Supprimer du texte

* x supprime le caractère situé sous le curseur
  Nx supprime N caractère à commencer par celui situé au niveau du curseur
  dw supprime le mot situé au niveau du curseur
  dNw supprime N à commencer par celui situé au niveau du curseur;
  e.g., d5w supprime 5 words
  D supprime l'ensemble des éléments entre le curseur et la fin de la ligne
* dd supprime/coupe la ligne où se situe le curseur
  Ndd or dNd delete N lines, beginning with the current line;
  e.g., 5dd deletes 5 lines

Couper et coller du texte

* yy copie la ligne actuelle
  Nyy or yNy copie les N prochaines lignes, incluant la ligne courante dans le presse papier
* p colle l'élément situé dans le presse-papier

Other Commands

Searching Text

A common occurrence in text editing is to replace one word or phase by another. To locate instances of particular sets of characters (or strings), use the following commands.

  /string search forward for occurrence of string in text
  ?string search backward for occurrence of string in text
  n move to next occurrence of search string
  N move to next occurrence of search string in opposite direction

Determining Line Numbers

Being able to determine the line number of the current line or the total number of lines in the file being edited is sometimes useful.

  :.= returns line number of current line at bottom of screen
  := returns the total number of lines at bottom of screen
  ^g provides the current line number, along with the total number of lines,
in the file at the bottom of the screen

Saving and Reading Files

These commands permit you to input and output files other than the named file with which you are currently working.

  :r filename<Return> read file named filename and insert after current line
(the line with cursor)
  :w<Return> write current contents to file named in original vi call
  :w newfile<Return> write current contents to a new file named newfile
  :12,35w smallfile<Return> write the contents of the lines numbered 12 through 35 to a new file named smallfile
  :w! prevfile<Return> write current contents over a pre-existing file named prevfile
