# Fiches validées
##

On trouve dans cette rubrique au niveau de l'onglet MesTâches (Admin-viseur),
	la liste des fiches validées.

	--> Il semble que la requête soit en erreur:
	select ti 
	from org.jbpm.taskmgmt.exe.TaskInstance as ti 
	where ti.actorId = :actorId   
	and ti.end is not null   
	and ti.name not like '%refus%'   
	and ti.isCancelled = false
		
	--> Cela renvoie la liste des fiches lancées et validées.
