    Notes pour le binding:
    ----------------------
    ExceptionForm prend en parametre 2 quadrants qui sont dans le formualire des id, et pour l'objetForm des objets quadrants.
			La syntaxe a respecté est donc la suivante:
			<html:select name="ExceptionForm" property="idQuadrant2.id">
				<html:optionsCollection name="lvbsQuadrant" />
			</html:select>
		  De cette manière on vient peupler l'id de l'objet quadrant.
		  Pour cela, il convient que l'ActionForm "ExceptionForm", ait dans un constructeur initialiser les objets à null pour 
		  pouvoir fair un setter dessus.
		  
		Notes sur l'enregistrement en base:
		-----------------------------------
		L'objet métier se voit affcter un id avec valeur 0, dans l'action à sa création.
		Problème: cela empêche l'enregistrement du bean en base.
		Solution: checker si id n'est pas null et pas égale à 0 sinon le mettre à nulle
